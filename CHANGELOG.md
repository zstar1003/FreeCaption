## [1.0.0] - 2025-10-10

### 新增
- ✨ 支持上传最多 9 张图片
- ✨ 第一张图片作为完整封面展示
- ✨ 后续图片自动截取字幕区域拼接
- ✨ 可调节字幕高度（50-500px）
- ✨ 支持顶部/底部字幕裁剪选择
- ✨ 生成长图预览功能
- ✨ 一键保存到相册
- ✨ 长按图片可保存
- ✨ 精美的渐变 UI 设计

### 技术特性
- 🔧 使用 TypeScript 开发
- 🔧 支持新旧两版 Canvas API
- 🔧 模块化代码结构
- 🔧 完善的错误处理
- 🔧 相册授权流程
