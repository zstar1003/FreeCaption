# 快速开始指南

## 5 分钟上手 FreeCaption

### 第 1 步：准备工作（1 分钟）

1. 下载微信开发者工具
   - 访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   - 下载对应平台版本并安装

2. 获取项目代码
   - 项目路径：`E:\code\FreeCaption`

### 第 2 步：导入项目（1 分钟）

1. 打开微信开发者工具
2. 选择"导入项目"
3. 填写以下信息：
   - 项目目录：选择 `E:\code\FreeCaption`
   - AppID：选择"测试号"（或填写自己的 AppID）
   - 项目名称：FreeCaption
4. 点击"导入"

###第 3 步：运行项目（1 分钟）

1. 项目导入后会自动编译 TypeScript
2. 等待编译完成（看不到错误提示即成功）
3. 在模拟器中查看效果

### 第 4 步：体验功能（2 分钟）

#### 添加图片
1. 点击界面中的"添加图片"按钮
2. 在弹出的对话框中选择图片
3. 可以添加 2-9 张图片

#### 调整参数
1. 拖动"字幕高度"滑块，调整截取高度
2. 选择"裁剪位置"（顶部或底部）
3. 实时预览图片列表

#### 生成长图
1. 至少添加 2 张图片后，"生成长图"按钮会激活
2. 点击按钮开始生成
3. 等待处理完成（显示"生成中..."）
4. 自动跳转到预览页面

#### 保存图片
1. 在预览页面查看生成的长图
2. 点击"保存到相册"按钮
3. 首次使用需要授权
4. 保存成功后显示提示

## 故障排除

### 问题 1：编译失败
**症状**：显示 TypeScript 编译错误
**解决**：
```bash
# 在项目目录下执行
npm install
npm run build
```

### 问题 2：Canvas 不显示
**症状**：点击生成长图没有反应
**解决**：
- 检查控制台是否有错误信息
- 确认图片已正确选择
- 尝试清除缓存重新编译

### 问题 3：无法保存到相册
**症状**：点击保存按钮没有反应或提示失败
**解决**：
- 检查是否授权相册访问权限
- 在模拟器中测试可能不支持，使用真机调试
- 确认图片路径有效

## 真机调试

### 预览到手机

1. 点击开发者工具右上角的"预览"按钮
2. 用手机微信扫描二维码
3. 在手机上体验小程序

### 真机调试

1. 点击开发者工具右上角的"真机调试"按钮
2. 用手机微信扫描二维码
3. 可以在开发者工具中查看手机上的调试信息
